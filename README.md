# روضة الأدب (Rawdat Al-Adab)

مشروع ويب متكامل لعرض نفائس الأدب العربي المجلوبة من قنوات تليجرام العامة، مع تصفية دقيقة للنصوص وتصميم أنيق يدعم الأوضاع المظلمة والمضيئة.

## المميزات
- **جلب تلقائي**: استخراج النصوص من 4 قنوات تليجرام أدبية رائدة.
- **تصفية صارمة**: إزالة الروابط، المعرفات، الأرقام، والرموز الزخرفية، مع الحفاظ على الإيموجي وتنسيق الشعر.
- **تصميم أنيق**: واجهة بسيطة ومريحة للعين باستخدام خط "Amiri" للأدب و "Great Vibes" للمطور.
- **عداد زوار حقيقي**: حساب الزوار الفريدين بناءً على عنوان IP (باستثناء localhost).
- **أوضاع العرض**: دعم كامل للوضع الليلي (Dark Mode) والوضع النهاري (Light Mode).

## المتطلبات التقنية
- Node.js (إصدار 14 أو أحدث)
- npm (مدير حزم Node)

## تعليمات الإعداد والتشغيل

### 1. الحصول على توكن بوت تليجرام (Telegram Bot Token)
1. افتح تطبيق تليجرام وابحث عن `@BotFather`.
2. أرسل الأمر `/newbot` واتبع التعليمات لتسمية بوتك.
3. سيقوم `BotFather` بتزويدك بـ `API Token`. قم بنسخه.

### 2. إعداد المشروع محلياً
1. قم بتحميل ملفات المشروع.
2. افتح المجلد في Terminal أو Command Prompt.
3. قم بتثبيت المكتبات اللازمة:
   ```bash
   npm install
   ```
4. افتح ملف `.env` وقم بوضع التوكن الخاص بك:
   ```env
   TELEGRAM_BOT_TOKEN=ضع_التوكن_هنا
   ```

### 3. تشغيل المشروع
1. قم بتشغيل السيرفر:
   ```bash
   node server.js
   ```
2. افتح المتصفح وانتقل إلى الرابط:
   `http://localhost:3000`

## هيكلية المشروع
- `server.js`: الواجهة الخلفية (Express) ومنطق جلب وتصفية البيانات.
- `public/`: ملفات الواجهة الأمامية (HTML, CSS, JS).
- `.env`: ملف الإعدادات والتوكن.
- `package.json`: ملف تعريف المشروع والمكتبات.

## ملاحظات تقنية
- يتم تخزين النصوص والزوار في الذاكرة المؤقتة (Memory Cache)، مما يعني أنها ستُفقد عند إعادة تشغيل السيرفر (بناءً على طلب عدم استخدام قاعدة بيانات).
- التصفية تستبعد أي نص يحتوي على روابط أو معرفات تواصل اجتماعي لضمان نقاء المحتوى الأدبي.
- يتم عرض نص عشوائي واحد عند كل تحميل للصفحة.
